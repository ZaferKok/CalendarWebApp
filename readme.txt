## Installation
- zip Datei (projekt_termincalender_zafer_kok.zip) entpacken.
- Ordner(projekt_termincalender_zafer_kok) unter XAMMP\htdocs ordner kopieren.
- XAMMP console starten und dann (apache/mySql) starten.
- http://localhost/phpmyadmin/ öffnen.
- eine Datenbank mit der Name "calendar" erstellen.
- unter Datenbank "calender", SQL (calendar.sql) importieren
- Die Datei befindet sich unter entpackte Ordner(calendar.sql).
- im Browser (Chrome) localhost/projekt_termincalender_zafer_kok Seite aufrufen.
